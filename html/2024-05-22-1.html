--- title: Windows11系统组件卸载，强迫症福音！ tags: Windows --- ## 卸载步骤 1. 同时按下Windows和R键，然后在运行命令框的文本框中键入powershell 2. 然后按Enter键，在系统上打开PowerShell窗口 3. 现在在 PowerShell 窗口中键入以下行，然后按Enter键 4. 演示代码 ```shell Get-AppxPackage Windows.DevHome | Remove-AppxPackage ``` 5. 执行此命令后，这个系统组件将从系统中删除 ## 卸载代码 1. Dev Home ```shell Get-AppxPackage Windows.DevHome | Remove-AppxPackage ``` 1. Game Bar ```shell Get-AppxPackage Microsoft.XboxGamingOverlay | Remove-AppxPackage ``` 1. Phone Link ```shell Get-AppxPackage Microsoft.YourPhone | Remove-AppxPackage ``` 1. 获取帮助 ```shell Get-AppxPackage Microsoft.GetHelp | Remove-AppxPackage ``` 1. 跨设备体验主机 ```shell Get-AppxPackage MicrosoftWindows.CrossDevice | Remove-AppxPackage ```
